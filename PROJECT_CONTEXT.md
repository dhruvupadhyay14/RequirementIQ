# PROJECT CONTEXT

RequirementIQ is an AI-powered Project Discovery Platform that integrates with Google Meet to analyze meeting transcripts, detect missing information, suggest follow-ups, and generate developer-ready documentation. This repository contains the initial production-ready monorepo scaffold.

Tech stack: React + Vite + TypeScript (apps/web), FastAPI + SQLAlchemy (apps/api), placeholder AI module (apps/ai), PostgreSQL, Docker, Docker Compose.

Follow the Development Guide in docs/ for instructions to run locally.
